# TeraTerm Macro Templates
TeraTermマクロ テンプレート集+α

## マクロテンプレート
### SerialConnect.ttl
シリアル接続し、root権限にログイン  
root権限へのログイン部分は場合により変更が必要

### SSHConnect_key.ttl
公開鍵認証でSSH接続

### SSHConnect_pwd.ttl
パスワード認証でSSH接続

## Tera Term設定
設定変更後、[設定の保存]からTERATERM.INIを上書き保存する必要あり
### 設定(全般)
[設定] > [その他の設定] > [全般]
- 「ブレーク送信のアクセラレーターキーを無効にする」にチェック
- 「ウィンドウサイズ変更時に表示内容をクリアする」のチェックを外す
### 設定(ログ)
[設定] > [その他の設定] > [ログ]
- 標準ログファイル名
  - %Y%m%d_%H%M%S.log
- 「自動的にログ採取を開始する」にチェック
- 「タイムスタンプ」にチェック
